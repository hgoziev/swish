# Swish

## Social App - react-native-app 

### 9 screens 

1. Onboarding 
2. Create Account 
3. Login 
4. Forgot Password
5. News Feed 
6. Shop 
7. Camera
8. Group Chat
9. Profile


### Getting Started 

1. git clone  or  Download zip file 

2. cd Project folder  &&  npm install 

3. npx react-native start  &&  npx react-native run-android
